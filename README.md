# jsphh-teaching

Teaching design skills for Claude Code.

## Installation

```bash
/plugin install github:jsphh7124/jsphh-teaching
```

## Skills

| Skill | Description |
|-------|-------------|
| `teaching-manual-builder` | 教學手冊建立（含範例章節） |
| `sketch-note-ppt` | 手繪風 PPT 設計 |
| `ppt-creator` | PPT 製作（含評分標準、需求表、檢查清單） |
