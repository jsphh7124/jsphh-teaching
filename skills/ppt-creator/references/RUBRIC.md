# PPT Creator - Quality Rubric

Score each item from 0-10. Total must be ≥ 75 to deliver. If score < 75, identify weakest 3 items and refine, then re-score (max 2 iterations).

## Scoring Criteria

### 1. Goal Clarity (0-10)
- **10**: Audience, objective, and CTA crystal clear and well-defined
- **7-9**: Most elements clear, minor ambiguity
- **4-6**: Some elements unclear or generic
- **1-3**: Vague or missing key elements
- **0**: No clear goal or audience

**Score**: ___

### 2. Story Structure (0-10)
- **10**: Perfect Pyramid structure, clear hierarchy, logical flow
- **7-9**: Good structure with minor gaps
- **4-6**: Structure present but weak or inconsistent
- **1-3**: Minimal structure, mostly list-like
- **0**: No discernible structure

**Score**: ___

### 3. Slide Assertions (0-10)
- **10**: All headings are testable assertion sentences with evidence
- **7-9**: Most headings are assertions, few topic labels
- **4-6**: Mix of assertions and topic labels
- **1-3**: Mostly topic labels, few assertions
- **0**: All topic labels, no assertions

**Score**: ___

### 4. Evidence Quality (0-10)
- **10**: Sufficient, credible, consistent data/cases/citations
- **7-9**: Good evidence with minor gaps
- **4-6**: Some evidence but incomplete or inconsistent
- **1-3**: Minimal or weak evidence
- **0**: No supporting evidence

**Score**: ___

### 5. Chart Fit (0-10)
- **10**: Perfect chart selection, complete labeling, highly readable
- **7-9**: Good charts with minor labeling issues
- **4-6**: Charts present but suboptimal type or incomplete labels
- **1-3**: Poor chart choices or missing critical labels
- **0**: No charts or completely inappropriate

**Score**: ___

### 6. Visual & Accessibility (0-10)
- **10**: Perfect contrast, font sizes, white space, AA compliant
- **7-9**: Good accessibility with minor issues
- **4-6**: Some accessibility issues present
- **1-3**: Multiple accessibility violations
- **0**: Fails basic accessibility standards

**Score**: ___

### 7. Coherence & Transitions (0-10)
- **10**: Natural, smooth transitions between all slides
- **7-9**: Good flow with occasional abrupt transitions
- **4-6**: Some awkward transitions
- **1-3**: Mostly disconnected slides
- **0**: No logical flow

**Score**: ___

### 8. Speakability (0-10)
- **10**: All notes 45-60 sec, natural language, easy to deliver
- **7-9**: Most notes good, some timing or language issues
- **4-6**: Notes present but awkward or wrong length
- **1-3**: Minimal or hard-to-speak notes
- **0**: No speaker notes

**Score**: ___

### 9. Deliverables Complete (0-10)
- **10**: All files present (slides.md, notes.md, refs.md, assets)
- **7-9**: Most files present, minor omissions
- **4-6**: Some key files missing
- **1-3**: Only basic files present
- **0**: Incomplete deliverables

**Score**: ___

### 10. Robustness (0-10)
- **10**: All gaps marked, fallback plans clear, next steps provided
- **7-9**: Most gaps addressed
- **4-6**: Some gaps acknowledged
- **1-3**: Minimal gap documentation
- **0**: No acknowledgment of limitations

**Score**: ___

## Evaluation Process

1. **Run CHECKLIST.md first** - Ensure all basic requirements met
2. **Score each item** - Be honest and objective
3. **Calculate total** - Sum all 10 scores
4. **If score ≥ 75**: Proceed to delivery
5. **If score < 75**: 
   - Identify the 3 lowest-scoring items
   - Write specific improvement actions for each
   - Apply improvements
   - Re-score (max 2 iterations total)
